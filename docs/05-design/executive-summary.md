# Executive Summary — CarRentalApp

## Название проекта
**CarRentalApp** — Мобильное приложение для аренды автомобилей

## Назначение
Разработка полнофункционального мобильного приложения для бронирования автомобилей,
обеспечивающего удобный интерфейс для клиентов, а также инструменты администрирования
для управления автопарком.

## Цели проекта
1. Реализовать мобильное приложение (Android Jetpack Compose) с современным UI
2. Разработать серверную часть (Spring Boot) с REST API
3. Обеспечить безопасность (JWT, BCrypt, ролевая модель)
4. Реализовать офлайн-режим (Room) для работы без интернета
5. Обеспечить качество кода (тесты >40%, SOLID, PCMEF)

## Функциональные возможности

### Для клиентов
- Регистрация и авторизация (JWT)
- Просмотр каталога автомобилей с фото и характеристиками
- Фильтрация по классу и датам
- Поиск по модели
- Бронирование автомобиля на выбранные даты
- Просмотр и отмена своих бронирований
- Офлайн-режим (кэширование данных)

### Для администраторов
- Панель управления автопарком
- Добавление новых автомобилей
- Изменение статуса автомобиля
- Статистика по автопарку

## Технический стек

| Компонент | Технология |
|-----------|------------|
| Клиент | Kotlin, Jetpack Compose, MVVM, Retrofit, Room |
| Сервер | Java 17, Spring Boot 3, Spring Security, JPA |
| База данных | PostgreSQL 16 |
| Аутентификация | JWT, BCrypt |
| Контейнеризация | Docker, Docker Compose |
| Документация API | Swagger / OpenAPI 3 |
| CI/CD | GitHub Actions |

## Архитектура
- **Backend:** Слоистая архитектура PCMEF (Presentation → Control → Mediator → Entity → Foundation)
- **Android:** MVVM (Model → View → ViewModel)
- **Паттерны GoF:** State, Strategy, Facade, Singleton

## Заключение
Разработанное приложение соответствует современным стандартам мобильной разработки,
включает полный цикл бронирования автомобилей, обеспечивает безопасность данных
и удобный интерфейс для всех категорий пользователей.
