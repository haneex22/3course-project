# Руководство менеджера

## 1. Назначение

Менеджер осуществляет контроль процесса аренды автомобилей, просматривает бронирования и контролирует состояние автопарка.

---

## 2. Вход в систему

Для работы необходимо выполнить вход под учетной записью менеджера.

### Учетная запись менеджера

**Логин:** `manager@carrent.ru`

**Пароль:** `password123`

После успешной авторизации открывается рабочая область менеджера.

---

## 3. Просмотр автомобилей

Менеджер может:

- просматривать весь каталог автомобилей;
- просматривать текущие статусы автомобилей;
- контролировать доступность автомобилей для аренды.

---

## 4. Работа с бронированиями

Менеджер имеет возможность:

- просматривать список бронирований;
- искать бронирования по пользователю или автомобилю;
- просматривать сведения о клиентах;
- контролировать текущие статусы бронирований.

---

## 5. Контроль состояния автопарка

Менеджер может просматривать автомобили со следующими статусами:

- AVAILABLE — свободен;
- RESERVED — забронирован;
- RENTED — передан клиенту;
- MAINTENANCE — находится на техническом обслуживании.

---

## 6. Поиск информации

Для поиска доступны следующие параметры:

- автомобиль;
- пользователь;
- статус автомобиля;
- статус бронирования.

---

## 7. Завершение работы

После окончания работы необходимо выполнить выход из системы через раздел **«Профиль»**.

Рекомендуется не оставлять открытую сессию без присмотра.
