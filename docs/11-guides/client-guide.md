# Руководство пользователя

## 1. Назначение системы

Мобильное приложение предназначено для поиска, просмотра и бронирования автомобилей, а также управления личным профилем пользователя. Приложение взаимодействует с серверной частью посредством REST API.

---

## 2. Требования

Для работы приложения необходимо:

- устройство под управлением Android 8.0 и выше;
- подключение к сети Интернет;
- зарегистрированная учетная запись.

---

## 3. Авторизация

После запуска приложения пользователю отображается экран авторизации.

Для входа необходимо:

1. Ввести адрес электронной почты.
2. Ввести пароль.
3. Нажать кнопку **«Войти»**.

При успешной авторизации пользователь автоматически переходит на главный экран приложения.

---

## 4. Регистрация

Если пользователь не зарегистрирован, необходимо:

1. Открыть экран регистрации.
2. Заполнить обязательные поля.
3. Нажать кнопку **«Зарегистрироваться»**.

После успешной регистрации становится доступен вход в систему.

---

## 5. Просмотр каталога автомобилей

Для просмотра автомобилей необходимо:

1. Открыть раздел **«Каталог»**.
2. Просмотреть список доступных автомобилей.
3. Выбрать интересующий автомобиль.

Карточка автомобиля содержит:

- марку;
- модель;
- стоимость аренды;
- текущий статус;
- описание.

---

## 6. Бронирование автомобиля

Для оформления бронирования необходимо:

1. Открыть карточку автомобиля.
2. Нажать кнопку **«Забронировать»**.
3. Выбрать даты аренды.
4. Подтвердить бронирование.

После успешного оформления информация сохраняется на сервере.

---

## 7. Личный профиль

Раздел **«Профиль»** позволяет:

- просматривать информацию о пользователе;
- изменять личные данные (если реализовано);
- просматривать историю бронирований.

---

## 8. Выход из системы

Для завершения работы необходимо:

1. Открыть раздел **«Профиль»**.
2. Нажать кнопку **«Выйти»**.

После выхода потребуется повторная авторизация.

---

## 9. Возможные ошибки


| Сообщение                 | Причина                     | Решение                           |
| ------------------------- | --------------------------- | --------------------------------- |
| Неверный логин или пароль | Ошибка авторизации          | Проверить введенные данные        |
| Автомобиль недоступен     | Автомобиль уже забронирован | Выбрать другой автомобиль         |
| Ошибка соединения         | Нет доступа к сети          | Проверить подключение к Интернету |
| Ошибка сервера            | Сервер временно недоступен  | Повторить попытку позже           |
