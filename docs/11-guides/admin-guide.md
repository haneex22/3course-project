# Руководство администратора

## 1. Назначение

Администратор осуществляет управление информационной системой проката автомобилей, включая управление автопарком, бронированиями и пользователями.

---

## 2. Вход в систему

Для входа в административную часть приложения необходимо использовать учетную запись администратора.

### Учетная запись администратора

**Логин:** `admin@carrent.ru`

**Пароль:** `password123`

После успешной авторизации открывается административная панель.

---

## 3. Управление автомобилями

Администратор может выполнять следующие действия:

- просматривать список автомобилей;
- добавлять новые автомобили;
- редактировать сведения об автомобилях;
- удалять автомобили;
- изменять статус автомобиля.

Доступные статусы автомобилей:

- AVAILABLE — доступен для аренды;
- RESERVED — забронирован;
- RENTED — находится в аренде;
- MAINTENANCE — находится на техническом обслуживании.

---

## 4. Управление бронированиями

Администратор имеет возможность:

- просматривать все бронирования;
- контролировать статусы бронирований;
- отменять бронирования при необходимости;
- отслеживать текущую загрузку автопарка.

---

## 5. Управление пользователями

Администратор может:

- просматривать список зарегистрированных пользователей;
- просматривать информацию о пользователях;
- контролировать роли пользователей;
- управлять учетными записями.

---

## 6. Рекомендации по безопасности

При работе рекомендуется:

- не передавать учетные данные третьим лицам;
- завершать сеанс после окончания работы;
- использовать административную учетную запись только для выполнения административных операций.

---

## 7. Завершение работы

После завершения работы необходимо открыть раздел **«Профиль»** и выполнить выход из системы с помощью кнопки **«Выйти»**.
